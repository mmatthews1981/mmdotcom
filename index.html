About Me

I'm a front-end developer. I write HTML, CSS and Javascript code that runs in browsers and publishes information on the internet for people to find. Everything else is implementation details.

Developer Axioms

  • All code is a liability.
  • Always hide with CSS what a lesser developer would use javascript to hide with CSS.
  • Programmers that optimize for keystrokes are dangerous.
  • If you're manipulating your DOM because that's where your state is you're probably doing it wrong.
  • Git is awful to work with, a finely-tuned expression of Linus Torvald's hatred of humanity, and I never want to do another project without it.
  • Pineapple is required on every pizza.
  • You cannot prevent your stylesheet from cascading.
  • Software that has launched is better than software that has not launched.
